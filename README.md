# Keybindings in simulator
keybinding | action
--- | ---
`W` | Move forward
`S` | Move backward
`A` | Move left
`D` | Move right
`Shift`| Move faster
`Ctrl` | Move slower
`Esc` | Exit simulator
`P` | Toggle pause
`M` | Toggle movement with mouse
`F` | Toggle follow bike mode
`N` | Next bike when in follow bike mode
`B` | Previous bike when in follow bike mode
